# USB监控程序 - 安装和使用说明

## 程序特性
1. **后台无窗口运行** - 完全隐藏，不打扰用户
2. **开机自启动** - 用户登录时自动启动
3. **U盘监控** - 自动检测U盘插入
4. **文件备份** - 根据关键词自动备份文件
5. **详细日志** - 记录所有操作到日志文件
6. **安全可靠** - 无需管理员权限，兼容杀毒软件

## 快速安装

### 方法一：直接运行（便携版）
1. 下载 `USBMonitor.exe`
2. 运行程序（无任何窗口）
3. 程序会在后台监控U盘

### 方法二：安装为服务
1. 下载所有程序文件
2. 运行 `USBMonitor.exe install`
3. 程序会添加到开机启动
4. 重启电脑后自动运行

### 方法三：使用安装程序
1. 运行 `USBMonitor_Setup.exe`
2. 按照向导完成安装
3. 程序会自动安装并启动
