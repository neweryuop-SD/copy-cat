# USB文件监控备份系统

一个智能的USB文件监控和备份程序，自动备份插入的USB设备中的重要文件。

## 主要功能

- ✅ **自动监控USB设备**：插入USB时自动检测
- ✅ **智能文件过滤**：按关键词和文件类型备份
- ✅ **磁盘空间管理**：空间不足时自动清理旧文件
- ✅ **开机自启动**：无需管理员权限
- ✅ **后台静默运行**：无窗口，不打扰用户
- ✅ **详细日志记录**：所有操作都有记录
- ✅ **可配置性强**：通过config.json轻松配置
